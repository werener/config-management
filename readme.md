# Инструмент визуализации графа зависимостей для менеджера пакетов

```https://github.com/werener/config-management```

## 1 ЭТАП: Конфигурация

Текущие функции:

    Возможность выбора файла конфигурации с помощью интерфейса командной
    Валидация параметров конфигурации:
        - Проверка на наличие обязательных параметров
        - Проверка на отсутствие несуществующих параметров
        - Проверка типов значений параметров
        - Валидация формата ссылок и версий
